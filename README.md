# 雷霆战机 - 关卡制飞机大战

一款经典的纵向卷轴射击游戏，类似雷霆战机风格。

## 游戏特色

- 6个精心设计的关卡
- 4种不同类型的敌机
- 道具系统（血量、武器、护盾、炸弹）
- 连击系统
- BOSS战
- 评级系统（S/A/B/C）

## 操作说明

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动飞机 |
| 空格键 | 释放全屏炸弹 |
| P | 暂停游戏 |
| R | 重新开始 |
| ESC | 退出游戏 |
| ENTER | 开始游戏 / 下一关 |

## 安装运行

### 1. 安装 Python
确保已安装 Python 3.7+

### 2. 安装依赖
```bash
pip install pygame
```

### 3. 运行游戏
```bash
python thunder_fighter.py
```

或者双击 `启动雷霆战机.bat`（Windows）

## 游戏说明

### 道具
- **红色 +** - HP回复30点
- **黄色 W** - 武器升级（最高5级）
- **青色 S** - 护盾（抵挡伤害）
- **橙色 B** - 炸弹+1

### 敌机类型
- **红色** - 普通敌机，速度慢
- **绿色** - 高速敌机，速度快但血少
- **橙色方块** - 坦克，血厚但速度慢
- **紫色大方块** - BOSS，发射扇形弹幕！

### 通关条件
- 第1-3关：消灭所有敌机即可通关
- 第4-6关：需要击败BOSS才能通关

### 评级系统
- **S** (5000分以上) - 传说级飞行员！
- **A** (3000-4999分) - 精英飞行员！
- **B** (1500-2999分) - 优秀飞行员！
- **C** (1500分以下) - 继续加油！
