# 雷霆战机 - 关卡制飞机大战

一款经典的纵向卷轴射击游戏，类似雷霆战机风格！

## 游戏截图

![游戏截图](https://img.shields.io/badge/点击运行-green)

## 游戏特色

- 6个精心设计的关卡
- 4种不同类型的敌机
- 道具系统（血量、武器、护盾、炸弹）
- 连击系统
- BOSS战
- 评级系统（S/A/B/C）

## 快速开始

### 方法一：直接下载运行（推荐）

1. 下载 [最新版本](https://github.com/0718lol/thunder-fighter/releases/latest)
2. 解压 ZIP 文件
3. 双击 `启动雷霆战机.bat` 即可开始游戏

### 方法二：从源码运行

**环境要求：**
- Python 3.7 或更高版本

**安装步骤：**

```bash
# 1. 克隆仓库
git clone https://github.com/0718lol/thunder-fighter.git

# 2. 进入目录
cd thunder-fighter

# 3. 安装依赖
pip install pygame

# 4. 运行游戏
python thunder_fighter.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动飞机 |
| 空格键 | 释放全屏炸弹（清屏！） |
| P | 暂停游戏 |
| R | 重新开始 |
| ESC | 退出游戏 |
| ENTER | 开始游戏 / 下一关 |
| H | 查看游戏说明 |

## 道具说明

| 道具 | 颜色 | 效果 |
|------|------|------|
| 回血 | 红色 + | HP回复30点 |
| 武器 | 黄色 W | 武器升级（最高5级） |
| 护盾 | 青色 S | 抵挡伤害 |
| 炸弹 | 橙色 B | 炸弹+1 |

## 敌机类型

| 类型 | 颜色 | 特点 |
|------|------|------|
| 普通敌机 | 红色 | 速度慢，数量多 |
| 高速敌机 | 绿色 | 速度快但血少 |
| 坦克 | 橙色方块 | 血厚但速度慢 |
| BOSS | 紫色大方块 | 发射扇形弹幕！ |

## 通关条件

- **第1-3关：** 消灭所有敌机即可通关
- **第4-6关：** 需要击败BOSS才能通关

## 评级系统

| 评级 | 分数要求 | 称号 |
|------|----------|------|
| S | 5000分以上 | 传说级飞行员！ |
| A | 3000-4999分 | 精英飞行员！ |

| C | 1500分以下 | 继续加油！ |

## 常见问题

**Q: 游戏闪退怎么办？**
A: 确保已安装 Python 和 Pygame，使用 `启动雷霆战机.bat` 启动游戏。

**Q: 中文显示乱码怎么办？**
A: 游戏已内置中文字体支持，如仍有问题请安装微软雅黑字体。

**Q: 如何自定义难度？**
A: 修改 `thunder_fighter.py` 中的 `LEVELS` 字典即可调整关卡配置。
